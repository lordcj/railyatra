## Environment Variables

Copy `.env.example` to `.env` and configure:

```env
VITE_RAILWAY_API_BASE_URL=https://indianrailapi.com/api/v2
VITE_ADSENSE_CLIENT_ID=ca-pub-YOUR_ACTUAL_ID
```
